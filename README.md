# **MindGear-iOS** 🎧🧠

# **"Mentale Stärke beginnt hier"** 💪🧘

> ⚠️ Dieses Projekt befindet sich derzeit in aktiver Entwicklung. Es dient als Teil meines Portfolios und wird kontinuierlich erweitert.

**MindGear** ist eine iOS-App, die speziell für **Men’s Mental Health** entwickelt wurde.  
Sie bietet eine kuratierte Sammlung inspirierender **YouTube-Videos & Podcasts** zu Themen wie:

- mentale Stärke  
- Persönlichkeitsentwicklung  
- emotionale Intelligenz  

Die App richtet sich an Männer, die sich aktiv mit ihrer mentalen Gesundheit auseinandersetzen möchten – egal ob Studierende 🎓, Berufstätige 💼 oder Sportler 🏋️.

## **✨ Features**

✅ **YouTube-Integration**  
→ Videos direkt in der App über **WebView** abspielen 📺

✅ **Favoriten-Funktion**  
→ Speichere inspirierende Inhalte für später ⭐

✅ **Tägliche Empfehlungen**  
→ Jeden Tag neue Inhalte entdecken 📅

✅ **Suchfunktion**  
→ Durchsuche Videos nach Themen & Kanälen 🔎

✅ **Dunkelmodus-Unterstützung**  
→ Für ein angenehmes Nutzungserlebnis 🌙

✅ **100% kostenfreie Entwicklung**  
→ Keine kostenpflichtigen APIs oder Drittanbieter notwendig 💸

## **🛠️ Technischer Aufbau**

### **📁 Projektstruktur**

Die App basiert auf dem **MVVM-Architektur-Muster** für eine saubere Trennung zwischen UI & Logik.

📁 **MindGear-iOS**  
┣ 📂 **Screens** – Startseite, Favoriten, Suche, Einstellungen  
┣ 📂 **Models** – Datenstruktur für Videos und Kanäle  
┣ 📂 **Services** – JSON-Handling & geplante YouTube-API-Logik  
┣ 📂 **Assets** – Farben, Icons, Schriftarten, Bilder  
┗ 📜 **README.md** – Diese Datei 😎

---

### **🖥️ Tech Stack**

🔹 **SwiftUI**  
→ Für das moderne User Interface

🔹 **SwiftData**  
→ Zur lokalen Speicherung von Favoriten

🔹 **WebKit (WebView)**  
→ Für das direkte Abspielen von YouTube-Inhalten

🔹 **JSON**  
→ Verwaltung der Videodaten & Kanalinformationen
